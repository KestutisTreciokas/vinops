# Vinops

Монорепозиторий: frontend/, backend/, общие скрипты и CI.

## Toolchain
- Node: >=20.14 <21 (`.nvmrc`=v20)
- Next: 14.2.5
- TypeScript: 5.6.x
- CI выводит версии и запускает lint/typecheck (см. `.github/workflows/toolchain-check.yml`).
